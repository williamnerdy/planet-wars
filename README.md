# Planet Wars [![Build Status](https://travis-ci.org/williamnerdy/planet-wars.svg?branch=master)](https://travis-ci.org/williamnerdy/planet-wars)

Url para visualizar a aplicação:

## https://williamnerdy.github.io/planet-wars

Para rodar a aplicação em modo local, instale o Node.js e npm/yarn

Baixe o repositório com o seguinte comando:

### git clone https://github.com/williamnerdy/planet-wars.git

Entre no diretório:

### cd planet-wars

Baixe o repositório e instale as dependências com o seguinte comando:

### yarn

ou

### npm install

No diretório da aplicação, execute:

### yarn start

ou

### npm start
